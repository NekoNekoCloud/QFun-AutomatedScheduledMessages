【使用前必读：初次配置指南】

开源仓库：https://github.com/NekoNekoCloud/QFun-AutomatedScheduledMessages
开源协议：MIT License （欢迎 Star、Fork 及遵守协议进行二次开发）

1、在 QFun 中点击启动本脚本，脚本将自动获取权限并生成默认配置文件（无需重启 QQ）。

2、配置文件生成后，请前往以下目录（该路径可在脚本菜单中复制，无需手动输入）：
示例路径：/storage/emulated/0/Android/data/com.tencent.mobileqq/QFun/你的QQ号/plugin/自动化定时消息/
请将“你的QQ号”替换为实际号码。若使用菜单复制功能，路径会自动替换为当前账号。

3、在该目录下，你可以通过两种方式自定义群号、发送时间和消息内容：
· 方式一：用文本编辑器直接修改 daily_config.json（每日定时）和 interval_config.json（间隔定时）。
· 方式二（推荐）：使用手机终端运行 manager.py 进行交互式管理。
运行前提：手机需已安装 Python 环境（如 Termux），进入配置目录后执行 python manager.py 即可。

【核心功能介绍】
 •每日定时发送：精确到秒的定时发送，跨天自动重置状态。
 •间隔循环发送：可设置时/分/秒间隔循环发送消息，内置防刷屏，最低间隔锁定为 30 秒。
 •断线自动补发：QQ 离线或后台被杀期间错过的消息，下次启动时会自动计算时间差并补发。
 •终端便捷管理：配套 Python 管理脚本，无需手动编辑 JSON，一键增删改任务。

【注意事项】
修改配置后，请在 QFun 界面中重新启动一次本脚本，使最新配置立刻生效。
如遇消息未发送，请查看该目录下的 log.txt 日志文件排查原因。
